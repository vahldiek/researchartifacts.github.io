# Charts Directory

SVG visualizations generated by `generate_visualizations.py`.

| Chart | Description |
|-------|-------------|
| `artifacts_by_year.svg` | Artifacts per year across all conferences |
| `conference_comparison.svg` | Side-by-side conference comparison |
| `conference_timeline.svg` | Timeline of conference artifact programs |
| `badge_distribution.svg` | Distribution of Available / Functional / Reproduced badges |
| `yearly_badge_trends.svg` | Badge trends over time |
| `systems_artifacts.svg` | Systems area artifact counts |
| `security_artifacts.svg` | Security area artifact counts |
| `total_artifacts.svg` | Combined total artifacts |

Regenerate with:
```bash
cd artifact_analysis
python generate_visualizations.py --data_dir ../researchartifacts.github.io
```
