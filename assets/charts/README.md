# Charts Directory

SVG visualizations generated by `generate_visualizations.py` and `generate_committee_stats.py`.

| Chart | Description |
|-------|-------------|
| `badge_distribution.svg` | Distribution of Available / Functional / Reproduced badges |
| `coverage_table.svg` | Coverage overview table |
| `systems_artifacts.svg` | Systems area artifact counts |
| `security_artifacts.svg` | Security area artifact counts |
| `total_artifacts.svg` | Combined total artifacts |
| `committee_countries_*.svg` | AE committee geographic distribution |
| `institution_*.svg` | Institution participation timelines |

Regenerate with:
```bash
cd reprodb-pipeline
python -m src.generators.generate_visualizations --data_dir ../reprodb.github.io
```
